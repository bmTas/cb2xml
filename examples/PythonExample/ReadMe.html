Python Examples

This directory contains example python programs that use the Xml Copybooks that cb2xml creates from Cobol copybooks. There are 2 python examples print_cb2xml.py and toTextCopybook.py. Both are designed to work with "xml copybooks" created with cb2xml

Important Files

  • cobol2xmlFileDescription.bat convert 2 COBOL copybooks to Xml copybooks so they can be used by the python programs.
  • print_cb2xml.py Details from a Cobol-Copybook like field position length
  • cobol2xmlFileDescription.bat and cobol2xmlFileDescription.sh Windows bat/shell scripts to run toTextCopybook.py.
  • toTextCopybook.py Take a binary (or zoned decimal) copybook and create 2 new Cobol copybooks:
    • Text Copybook a new copybook where all fields are plain text
    • Code Copybook a copybook that moves fields from the original copybook to the output copybook
