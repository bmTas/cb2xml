Examples Directory

As you might expect, the sub-directories in this directory all hold various examples of Converting Cobol Copybooks to Xml and processing Cobol Copybooks / cb2xml-Xml in various languages. In particular:
  • RunningCb2xml holds bat/shell script examples of converting Cobol-Copybooks to Xml + converting Cobol-Data files To / from Xml. If you want to convert Cobol data files to/from Xml you are better of using JRecord.
  • The GroovyExample, JRubyExample and JythonExample hold programs that process Cobol-Copybooks / cb2xml-Xml using jaxb.
  • The RubyExample and PythonExample directories hold example programs that read cb2xml generated Xml.
  • Finally JavaExample holds bat files to run the java example programs
