Jython Examples

The main differences between Jython code and Python code are:

  • In Jython we can process the Cobol Copybooks directly and not just the Xml produced by cb2xml
  • You can also convert the Xml/Copybook into specific Java classes using JAXB.
  • The Jaxb code is in the cb2xml_Jaxb.jar

The 2 example programs are:

  • print_cb2xml.py - prints a cb2xml Xml copybook.
  • print_Cobol.py - prints a Cobol copybook.
